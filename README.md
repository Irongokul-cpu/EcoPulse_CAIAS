## 📋 Overview

**EcoPulse** is a comprehensive ESG (Environmental, Social, Governance) Intelligence Platform that provides real-time climate monitoring, sustainability metrics, and environmental data for major Indian cities.

## ✨ Features

| Module | Description |
|--------|-------------|
| 🗺️ Climate Map | Google Maps with 15+ cities |
| 📊 ESG Dashboard | Environmental, Social, Governance scores |
| 🌤️ Climate Monitor | Weather and 10-day forecast |
| ⚡ Energy Management | Track consumption and emissions |
| ♻️ Waste Management | Recycling stats and composition |
| 📉 Carbon Footprint | CO₂ tracking and trends |
| 🌱 Smart Agriculture | Crop health monitoring |
| 💧 Water Conservation | Usage metrics and tips |
| 🛰️ Live Satellite | ISRO feed simulation |
| 🏆 Leaderboard | Community rankings |
| 👤 Profile | User statistics |
| 🤖 AI Assistant | Climate chatbot |
