## 📋 Overview

**EcoPulse** is a comprehensive ESG (Environmental, Social, Governance) Intelligence Platform that provides real-time climate monitoring, sustainability metrics, and environmental data for major Indian cities. This single-page application delivers professional-grade insights through an intuitive dark-themed interface.

---

## ✨ Key Features

| Module | Description |
|--------|-------------|
| 🗺️ **Climate Map** | Google Maps integration with 15+ Indian cities |
| 📊 **ESG Dashboard** | Environmental, Social & Governance scores |
| 🌤️ **Climate Monitor** | 10-day forecasts and real-time weather data |
| ⚡ **Energy Management** | Track consumption, costs, and carbon emissions |
| ♻️ **Waste Management** | Recycling stats and waste composition |
| 📉 **Carbon Footprint** | City-level carbon tracking |
| 💧 **Water Conservation** | Usage metrics and conservation tips |
| 🛰️ **Live Satellite** | ISRO satellite feed simulation |
| 🏆 **Leaderboard** | Community rankings and achievements |
| 🤖 **AI Assistant** | Smart chatbot for climate queries |
